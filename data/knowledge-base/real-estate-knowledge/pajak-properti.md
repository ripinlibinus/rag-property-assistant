# Panduan Pajak Properti di Indonesia

## Pajak Saat Membeli Properti

### 1. BPHTB (Bea Perolehan Hak atas Tanah dan Bangunan)

**Pengertian:** Pajak yang dibayar oleh PEMBELI saat memperoleh hak atas tanah/bangunan.

**Rumus:**
```
BPHTB = 5% x (NPOP - NPOPTKP)

NPOP = Nilai Perolehan Objek Pajak (harga transaksi atau NJOP, mana lebih tinggi)
NPOPTKP = Nilai Perolehan Objek Pajak Tidak Kena Pajak (berbeda tiap daerah)
```

**Contoh Perhitungan:**
```
Harga rumah: Rp 1,000,000,000
NJOP: Rp 800,000,000
NPOP (tertinggi): Rp 1,000,000,000
NPOPTKP Medan: Rp 80,000,000

BPHTB = 5% x (1,000,000,000 - 80,000,000)
BPHTB = 5% x 920,000,000
BPHTB = Rp 46,000,000
```

**NPOPTKP per Kota (Referensi):**
- Jakarta: Rp 350 juta (rumah pertama)
- Surabaya: Rp 75 juta
- Medan: Rp 80 juta
- Bandung: Rp 80 juta

*Cek perda terbaru karena bisa berubah

---

### 2. PPN (Pajak Pertambahan Nilai)

**Kapan dikenakan:**
- Membeli dari developer (properti baru)
- Tidak dikenakan untuk jual beli antar individu

**Tarif:** 11% dari harga jual

**Catatan:**
- Developer biasanya sudah include PPN di harga
- Tanyakan apakah harga "sudah termasuk PPN" atau belum

---

## Pajak Saat Menjual Properti

### 1. PPh Final (Pajak Penghasilan)

**Pengertian:** Pajak yang dibayar oleh PENJUAL atas penghasilan dari penjualan properti.

**Tarif:** 2.5% dari harga jual (bruto)

**Contoh:**
```
Harga jual: Rp 1,000,000,000
PPh Final = 2.5% x 1,000,000,000
PPh Final = Rp 25,000,000
```

**Syarat:**
- Dibayar sebelum AJB (Akta Jual Beli)
- Bukti bayar (SSP) dilampirkan di AJB

**Pengecualian:**
- Warisan (bukan penjualan)
- Hibah ke keluarga sedarah

## Pajak Tahunan

### PBB (Pajak Bumi dan Bangunan)

**Pengertian:** Pajak tahunan atas kepemilikan tanah dan bangunan.

**Rumus:**
```
PBB = 0.5% x NJKP

NJKP = Nilai Jual Kena Pajak
     = 20% x NJOP (jika NJOP < 1M)
     = 40% x NJOP (jika NJOP ≥ 1M)
```

**Contoh:**
```
NJOP: Rp 500,000,000
NJKP = 20% x 500,000,000 = Rp 100,000,000
PBB = 0.5% x 100,000,000 = Rp 500,000

PBB Tahunan = Rp 500,000
```

**Pembayaran:**
- Tagihan keluar setiap awal tahun
- Jatuh tempo biasanya Agustus
- Bayar di bank/Tokopedia/Shopee

---

## Pajak Sewa Properti

### PPh atas Sewa

**Tarif:** 10% dari nilai sewa (final)

**Contoh:**
```
Sewa rumah per tahun: Rp 60,000,000
PPh Sewa = 10% x 60,000,000 = Rp 6,000,000

Diterima pemilik: Rp 54,000,000
```

**Catatan:**
- Dipotong oleh penyewa (jika badan usaha)
- Dilaporkan sendiri jika penyewa individu

## Summary Pajak Properti

| Jenis Pajak | Tarif | Dibayar Oleh | Kapan |
|-------------|-------|--------------|-------|
| BPHTB | 5% x (NPOP-NPOPTKP) | Pembeli | Saat beli |
| PPN | 11% | Pembeli | Saat beli (developer) |
| PPh Final | 2.5% | Penjual | Saat jual |
| PBB | 0.5% x NJKP | Pemilik | Tahunan |
| PPh Sewa | 10% | Pemilik | Saat terima sewa |

## Tips untuk Agent

### Saat Negosiasi:
1. **Jelaskan biaya pajak ke kedua pihak**
   - Buyer: BPHTB, PPN (jika baru)
   - Seller: PPh Final

2. **Negosiasi siapa bayar apa**
   - Default: Masing-masing bayar pajaknya
   - Bisa dinegosiasi split atau ditanggung satu pihak

3. **Perhitungkan di total biaya**
   - Buyer perlu siapkan: DP + BPHTB + biaya notaris + biaya KPR
   - Seller terima bersih: Harga - PPh - komisi agent

### Hitung Cepat untuk Buyer:
```
Persiapan dana = Harga + 7-10%

Contoh: Rumah 1M
- DP 20%: 200jt
- BPHTB ± 5%: 50jt
- Notaris ± 1.5%: 15jt
- KPR (provisi dll) ± 2%: 20jt
Total: Rp 285 juta (bukan cuma 200jt DP!)
```

### Dokumen Pajak yang Diperlukan:
- SSP (Surat Setoran Pajak) PPh
- SSP BPHTB
- SPPT PBB tahun berjalan (lunas)
- Bukti bayar PBB 3-5 tahun terakhir

## FAQ Pajak

### "NJOP di mana lihatnya?"
- Di SPPT PBB
- Atau cek di website pemda

### "BPHTB bayar kapan?"
- Sebelum AJB di notaris
- Bayar via bank atau e-BPHTB

### "PPh siapa yang bayar?"
- Tanggung jawab penjual
- Tapi bisa dinegosiasi

### "Kalau warisan kena pajak?"
- Tidak kena PPh (bukan penjualan)
- Tapi tetap kena BPHTB dengan NPOPTKP khusus warisan (lebih tinggi)
